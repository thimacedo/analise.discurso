# Description

## Problem to solve
A latência agregada dos workers precisa ser apresentada de forma clara e visualmente atraente no dashboard.

## Solution
Criar um componente frontend (ex: tabela ou cards) que consome o endpoint de API de latência agregada e exibe as métricas de forma legível.

## Implementation Details
- Consumir o endpoint de API de latência agregada.
- Renderizar dados em formato tabular ou de cartões.
- Garantir que a exibição seja responsiva.
