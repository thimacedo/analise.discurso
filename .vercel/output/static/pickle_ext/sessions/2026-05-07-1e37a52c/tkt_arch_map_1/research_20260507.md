# Research: Mapeamento da Arquitetura Sentinela Democrática

**Date**: 2026-05-07

## 1. Executive Summary
A arquitetura v18.5+ do projeto Sentinela Democrática é baseada em um pipeline de dados distribuído, coordenado por um Orquestrador Central. O sistema utiliza múltiplos motores de coleta, processamento forense (PASA v16.4) e uma API FastAPI que serve um dashboard rico em JS.

## 2. Technical Context
### 2.1 Fluxo de Dados (Ingestion to Delivery)
1. **Coleta**: 
    - `core/instagram_headless.py` (Playwright) e `sentinela_scraper/spiders/instagram.py` (Scrapy) capturam comentários.
    - `core/meta_ad_scraper.py` monitora a biblioteca de anúncios da Meta.
2. **Processamento Forense**:
    - `core/orquestrador.py` consome a fila do Supabase.
    - `core/ai_service.py` e `core/pasa_auditor.py` executam a classificação forense baseada no Protocolo PASA v16.4.
    - `processing/data_miner.py` realiza clustering (KMeans) para detecção de redes coordenadas.
3. **Persistência**: 
    - Supabase (PostgreSQL) com RLS ativado. Tabelas principais: `candidatos`, `comentarios`, `alertas_ativos`.
4. **Entrega (API & UI)**:
    - `api/index.py` (FastAPI) expõe os endpoints de métricas e alertas.
    - `src/core/ui.js` e `src/core/app.js` renderizam o dashboard.

## 3. Findings & Analysis
- **Orquestrador Central**: O arquivo `core/orquestrador.py` é o componente mais crítico, gerenciando o ciclo de vida dos dados.
- **Protocolo PASA**: Centralizado em `core/pasa_auditor.py` e `core/forensics_service.py`, garantindo o rigor criminal na análise.
- **Infraestrutura de Alertas**: Integração com Firebase em `core/firebase_alerter.py`.

## 4. Technical Constraints
- O sistema depende fortemente da disponibilidade do Supabase e de tokens de IA para o PASA.
- A coleta via Playwright Headless exige manutenção constante de seletores.

## 5. Architecture Documentation
- O sistema segue o padrão **Worker-Centric**, onde cada tarefa é processada de forma idempotente.
- Padrão de projeto: **Composition over Inheritance** visível no uso de `BaseWorker`.
